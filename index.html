This Site

Based on the HTML5 Boilerplate and the twitter bootstrap this is an attempt at following best practices in web design. It aims to be as enjoyable as possible to code, whilst still being beautiful, user friendly and screamingly fast.

Coffeescript and Less make things easy to code, and that means I can spend more time on the fun bits. The bootstrap framework means that I can make things pretty, even though I have exactly no flair for design. As you can tell, its early days, have a look on github if your interested.
